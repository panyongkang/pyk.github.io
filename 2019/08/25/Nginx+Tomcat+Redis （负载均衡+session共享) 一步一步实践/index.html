Nginx+Tomcat+Redis （负载均衡+session共享) 一步一步实践
Created 2019-08-25| Updated 2019-09-12|LinuxNginxTomcatRedis

简言

前面我们实践过简单的Nginx+Tomcat反向代理负载均衡，在学习的过程中发现企业级的应用中大都会和Redis结合起来使用。这里我们使用测试环境实践一下。下面记录过程。正好我们之前的实践是在linux上开发的，这里我们在window上安装nginx来体验一下负载均衡+session共享！

一、准备工作

操作系统：Windowserver2016

反向代理：Nginx_1.4.7

Tomcat：pache-tomcat-7.0.42

复制三份：

apache-tomcat-7.0.42_A;

apache-tomcat-7.0.42_B;

apache-tomcat-7.0.42_C

Redis:redis-64.3.0.503

JDK:1.8
