好用的图床工具推荐-PicGo+GitHub图床，让Markdown飞
Created 2019-09-08| Updated 2019-09-10|GitHub

简言

所谓图床工具，就是自动把本地图片转换成链接的一款工具，网络上有很多图床工具，就目前使用种类而言，PicGo 算得上一款比较优秀的图床工具。它是一款用 Electron-vue 开发的软件，可以支持微博，七牛云，腾讯云COS，又拍云，GitHub，阿里云OSS，SM.MS，imgur 等8种常用图床，功能强大，简单易用。
我这里之前使用的是七牛云+Mpic的组合，不过中间七牛云测试域名收回之后我的图床就瘫痪了，所有的图片字资源也随之付之东流，很是生气。思考了一段时间，就像自己去搭建一个图床服务，不过这玩意成本那么高，备案+购买服务器一年好几百块。而且也没有必要去花这个钱，偶尔在网上看到用GitHub+PicGo的方式搭建图床服务，哇塞这个也太棒了吧！666这里记录一下搭建的教程以供大家参考。

一、GitHub仓库配置
